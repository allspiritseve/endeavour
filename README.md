# Endeavour

Try, but better.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'endeavour'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install endeavour

## Usage

```ruby
car = nil
car.try.drive # => nil
```
